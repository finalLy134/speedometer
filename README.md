# speedometer
A simple FiveM standalone script that adds speedometer UI.

## Features
- Shows vehicle speed
- Optimized
- Beautiful UI
- Drag and Drop

## Configuration
In order to change from KPH to MPH or MPH to KPH simply go into the `config.json` and change the `measureUnits` to the type you wish.
